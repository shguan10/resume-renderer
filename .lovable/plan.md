
## Hello, World — Precision Landing Page

A refined "Hello, World" page with subtle interactivity, establishing a polished design foundation.

### What we'll build:
1. **Typography**: Import Geist font, fluid responsive sizing with `clamp()`, tight letter spacing
2. **Color system**: Cool near-white background (`hsl(240 10% 98%)`), soft charcoal text, warm gold accent
3. **Cursor light effect**: A radial gradient that follows the mouse, adding subtle depth
4. **Entry animation**: Text fades and slides up on page load using CSS animation (no extra dependencies)
5. **Hover interaction**: Letter spacing subtly expands on hover for a "breathing" feel

Single centered `h1` on a full-viewport page — minimal elements, refined execution.
